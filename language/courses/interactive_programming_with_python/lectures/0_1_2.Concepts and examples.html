    Concepts and examples Help Center Learn more.

    This page organizes the topics covered in class by week. Each week's page includes a short summary of the topic and also includes links to examples from lectures plus more supplemental examples. Note that the concepts in the first couple of weeks are described in greater depth to aid novice programmers learning to program in Python.


    • Week 0 - Comments, strings, numbers, arithmetic operators, arithmetic expressions, variables
    • Week 1 - Functions, indentation, remainders, modules, Boolean expressions, relational operators, conditional statements
    • Week 2 - Event handlers, local variables, global variables, SimpleGui module, buttons, input fields

    If you are having trouble understanding a particular message in Python, Emily Wachtel has made a nice guide to understanding Python errors.
